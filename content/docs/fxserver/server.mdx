### `serverStop`

Gracefully shuts down the running server environment.

**Payload:**
<TypeTable
  type={{
    by: {
      type: 'number',
      required: false,
      description: 'The database ID of the Admin/Staff member initiating the server shutdown.',
    },
  }}
/>

```lua tab="Lua"
local by = 1
local response = exports.fxManager:serverStop(by)

if response.ok then
    print("Server stop command processed successfully.")
else
    print("Failed to stop server:", response.error)
end
```

```js tab="JS"
const by = 1;
const response = await exports.fxManager.serverStop(by);

if (response.ok) {
  console.log("Server stop command processed successfully.");
} else {
  console.error(`Failed to stop server: ${response.error}`);
}
```

**Payload:**
<TypeTable

      required: false,

/>

```lua tab="Lua"
local by = 1
local response = exports.fxManager:serverRestart(by)

else
    print("Failed to restart server:", response.error)
end
```

```js tab="JS"
const by = 1;
const response = await exports.fxManager.serverRestart(by);

if (response.ok) {
```
